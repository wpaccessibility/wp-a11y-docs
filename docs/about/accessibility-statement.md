# Accessibility statement

For the website wpaccessibility.org and all its content we aim to meet the latest recommended version of the Web Content Accessibility Guidelines (WCAG) level AA: [WCAG 2.2 AA](https://www.w3.org/WAI/WCAG22/quickref/).

In January 2026 wpaccessibility.org was officially audited and no accessibility issues were found: [Evaluation Report for WPAccessibility.org](https://audits.level-level.com/wp-accessibility/), by [Level Level Platforms & Accessibility](https://level-level.com/).

This website is a work in progress. Did you find an accessibility issue or have a problem using the website? Please [contact us]({{site.baseurl}}/docs/contact/).

{: .callout  .info }
**Please note:** this Accessibility statement is for **this website** and not for WordPress itself. If you find an accessibility issue in WordPress core, or in one of the core themes, please create a [new ticket on Make WordPress Core](https://core.trac.wordpress.org/newticket) and give it the focus “accessibility”.
