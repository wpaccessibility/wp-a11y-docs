# Theme guidelines for the WordPress accessibility-ready program

Theme developers may request an optional accessibility review as part of the theme review process or later for an existing theme.  Submitted themes (or theme updates) that pass a manual review are approved to be `accessibility-ready` and show up in the theme directory as accessibility-ready.

This section outlines the `accessibility-ready` requirements with testing instructions for each requirement.

For additional information about the accessibility ready testing process, see:

* [Theme developer handbook: Accessibility](https://make.wordpress.org/themes/handbook/review/accessibility/)
* [Accessibility-ready testing process]({{site.baseurl}}/docs/accessibility-ready/testing-themes/)
